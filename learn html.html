HTML (HyperText Markup Language) is the backbone of all websites. It defines the structure and content of web pages.

1. Basic HTML Page Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

2. Headings & Paragraphs

HTML provides 6 levels of headings, from <h1> (most important) to <h6>.


<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph of text.</p>

3. Lists

You can create ordered or unordered lists:


<ul>
  <li>Item One</li>
  <li>Item Two</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

4. Links

Links use the <a> tag with an href attribute:


<a href="https://www.example.com">Visit Example</a>

5. Images


<img src="images/example.jpg" alt="Example image" width="400">

6. Tables


<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>30</td></tr>
  <tr><td>Bob</td><td>25</td></tr>
</table>

7. Forms


<form action="/submit" method="post">
  <label>Name: <input type="text" name="name"></label><br>
  <input type="submit" value="Submit">
</form>
