1. Basic HTML Page Structure

üìñ Understanding HTML Structure

HTML (HyperText Markup Language) forms the foundation of web pages. It consists of elements that define content, structure, and formatting.

Every HTML page follows a standard structure:


<!DOCTYPE html>  <!-- Declares the document type -->
<html>           <!-- Root element of the page -->
  <head>        <!-- Metadata & page title -->
    <title>My First Page</title>
  </head>
  <body>        <!-- Visible content -->
    <h1>Welcome to HTML!</h1>
    <p>HTML structures your webpage.</p>
  </body>
</html>

    Key Elements Explained:
  • <html>- is the root element, wrapping everything
  • <head>- contains metadata, title, and links to styles
  • <body>- holds the visible content displayed in browsers
  • <h1> to <h6>-define headings for structure
  • <p>- creates paragraphs for readable text

Understanding this structure ensures proper rendering, SEO optimization, and page accessibility.

2. Headings & Paragraphs

Headings are one of the most fundamental elements in web development. Every well-structured webpage relies on headings to guide the user through information efficiently.

Heading serve as:

  • Visual Hierarchy: They create a clear structure, making the page easy to scan
  • SEO Optimization: Search engines use headings to understand the page's topic
  • Accessibility: Screen readers rely on headings to help visually impaired users navigate a website
There are a total of six (6) headings, h1-h6. Each heading has a key role it plays in the website structure.

Paragraphs

The <P> tag in HTML defines a paragraph, which is a fundamental block-level element used to organize and display text content on a webpage. It creates distinct sections of text, separating ideas logically and enhancing readability.

3. Lists

Lists in HTML: Lists are essential for organizing information, making it more structured and easy to read. They allow content to be presented in a clear, sequential or unordered format, ensuring readability and logical flow.
There are various types of lists, but we will be focusing on two: Ordered and Unordered Lists.

1Ô∏è‚É£ Ordered Lists <ol>: An ordered list is used when items need a specific order. They are often numbered or lettered, making them suitable for step-by-step instructions, rankings, or sequences.

2Ô∏è‚É£ Unordered Lists <ul>: These lists do not follow a specific sequence. They are commonly used for presenting items that do not need numbering, such as navigation menus, features, or bullet points in articles.

Sample:

  <li>Item One</li>
  <li>Item Two</li>
</ul>

</ol>

4. Links

Links, created using the <a> tag, allow users to navigate between web pages, external sites, and resources. They are the backbone of web navigation, making content accessible across different locations. 

<a href="https://www.example.com">Visit Example</a>

5. Images

Images are defined by the <img> tag, visually enhance web content and provide meaningful illustrations that engage users.

<img src="images/example.jpg" alt="Example image" width="400">

6. Tables

Tables in HTML provide a structured format for displaying information in a clear, easy-to-read way. By organizing data into rows and columns, tables make it simple to compare values, showcase statistics, and present structured content effectively. 

<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Alice</td><td>30</td></tr>
  <tr><td>Bob</td><td>25</td></tr>
</table>
