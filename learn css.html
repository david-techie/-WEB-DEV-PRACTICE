1. CSS Selectors

CSS selectors allow you to pinpoint and style specific HTML elements, whether you're working with simple tag selectors, class selectors, or more complex combinations. By mastering selectors, you gain full control over your webpage's styling ensuring elements look exactly how you want, from subtle adjustments to bold transformations.


/* Select all paragraphs */
p {
  color: blue;
}
/* Select element by ID */
#main-title {
  font-size: 24px;
}
/* Select elements by class */
.card {
  border: 1px solid #ccc;
}

2. Colors & Backgrounds

Color sets the mood, creates contrast, and makes designs pop, Whether you're using named colors, hex codes, or RGB/HSLA values, mastering color application is key to crafting visually stunning and accessible web designs. Combine gradients, transparency, and strategic color palettes to enhance depth and style, because the right colors don't just decorate, they communicate. 


body {
  background-color: #f9f9f9;
  color: #333;
}
div {
  background: linear-gradient(to right, red, yellow);
}

3. Fonts & Text

Typography or Text is more than just letters, it's how you shape readability, style, and personality in your design. With custom fonts, sizes, and spacing, you can create text that's bold, elegant, or playful, depending on your vision. Font families define the tone, sizes establish hierarchy, and spacing ensures balanceâ€”all working together to make content engaging and easy to read.

  font-family: 'Segoe UI', sans-serif;
  font-size: 32px;
  letter-spacing: 1px;
  text-align: center;

4. Box Model

The CSS Box Model defines how elements are sized and spaced on a webpage.Every element is treated as a box with four key components:

  1. Content: The actual text or image inside the element
  2. Padding: Space between the content and the border, ensuring breathing room
  3. Border: The visible or invisible edge that wraps the content
  4. Margin: Space outside the border, controlling the distance between elements


.card {
  margin: 20px;
  padding: 10px;
  border: 1px solid #ccc;
}

5. Layout: Flexbox & Grid

Flexbox and Grid are powerful CSS layout systems designed to create responsive, dynamic, and well-structured designs with ease. Flexbox excels at aligning and distributing elements efficiently, while CSS Grid offers a more structured approach for complex, multi-dimensional layouts. Together, they enable developers to build adaptive, visually appealing websites that seamlessly adjust to different screen sizes.

Flexbox


.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Grid


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}
