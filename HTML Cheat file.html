HTML Cheat Sheet - Common Tags

Tag Description Example
<html> Defines the root of an HTML document <html>...</html>
<head> Contains metadata and links to styles/scripts <head>...</head>
<title> Defines the document title <title>My Page</title>
<body> Defines the body of the document <body>...</body>
<h1> to <h6> Headings from largest to smallest <h1>Welcome</h1>
<p> Defines a paragraph <p>Hello world</p>
<a> Creates a hyperlink <a href="https://example.com">Click me</a>
<img> Embeds an image <img src="image.jpg" alt="Image">
<ul><ol><li> Creates lists <ul><li>Item 1</li></ul>
<table><tr><td> Defines a table structure <table><tr><td>Data</td></tr></table>
<form><input><button> Forms and input fields <form><input type="text"><button>Submit</button></form>
<div><span> Defines containers for styling <div>Block</div> <span>Inline</span>
<iframe> Embeds another document inside <iframe src="page.html"></iframe>
<script> Defines client-side JavaScript <script>alert('Hello!');</script>
