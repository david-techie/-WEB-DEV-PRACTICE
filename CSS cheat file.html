CSS Cheat Sheet - Common Properties

Property Description Example
color Sets the text color color: red;
background-color Sets the background color background-color: blue;
font-size Defines the text size font-size: 16px;
font-family Defines the font type font-family: 'Arial', sans-serif;
margin Defines space around an element margin: 10px;
padding Defines space inside an element padding: 20px;
border Defines border style border: 1px solid black;
width Defines element width width: 80%;
height Defines element height height: 200px;
display Controls element visibility display: none;
position Defines element positioning position: absolute;
text-align Aligns text horizontally text-align: center;
z-index Defines layering order z-index: 10;
box-shadow Adds a shadow to an element box-shadow: 4px 4px 8px rgba(0, 0, 0, 0.3);
transition Animates CSS property changes transition: all 0.3s ease-in-out;
animation Applies animations animation: bounce 2s infinite;
